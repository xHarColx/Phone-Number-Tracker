# Contributing to PhoneTrackerPro

Thanks for your interest in contributing! 🎉

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork locally
3. **Create a branch** for your feature: `git checkout -b feature/my-feature`
4. **Make your changes** and test them
5. **Commit** with a clear message: `git commit -m "feat: add X feature"`
6. **Push** to your fork: `git push origin feature/my-feature`
7. **Open a Pull Request** against `main`

## Guidelines

- Keep code style consistent with existing codebase
- Add comments for complex logic
- Test your changes before submitting
- Update README.md if adding new features or CLI flags
- Do NOT commit `.env` files or API keys
- Do NOT commit `output/` directory contents

## Reporting Issues

- Use GitHub Issues
- Include your Python version, OS, and error traceback
- Describe what you expected vs what happened

## Code of Conduct

Be respectful. This is an educational tool — contributions should align with ethical security research.
